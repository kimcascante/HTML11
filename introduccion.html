Introducción

El objetivo de este proyecto es demostrar los conocimientos adquiridos en el curso de HTML, mediante el desarrollo de un sitio web interactivo. Durante el curso, se abordaron conceptos fundamentales de HTML5, CSS3, y técnicas de diseño web responsivo, que se aplican en este proyecto.

Problema Identificado:

    En el contexto del curso, se identificó la necesidad de desarrollar un proyecto que no solo aplicara las estructuras básicas de una página web, sino que también demostrara habilidades en la integración de elementos interactivos y multimedia, así como la optimización para dispositivos móviles.

Este sitio web tiene como objetivo proporcionar una experiencia de usuario fluida y moderna, a la vez que cumple con los requisitos técnicos establecidos en el plan de estudios del curso.
