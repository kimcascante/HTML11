Página de Requerimientos

El objetivo principal de este proyecto es crear un sitio web interactivo que demuestre la aplicación de los conceptos aprendidos en el curso de HTML.
Desarrollar un sitio web interactivo utilizando HTML5 y CSS3, que incluya elementos multimedia, formularios y un diseño responsivo, aplicando las mejores prácticas en desarrollo web.

Requerimientos Funcionales:

  1. Interactividad: El sitio debe incluir formularios interactivos como formularios de contacto o encuestas.
  2. Responsividad: El diseño debe ser adaptable a diferentes dispositivos (escritorios, tablets, móviles).
  3. Elementos Multimedia: Incluir videos, audios y/o animaciones HTML5.
  4. Optimización SEO: Aplicación de principios básicos de SEO, como el uso adecuado de títulos, meta descripciones, y etiquetas alt para imágenes.

Requerimientos de Diseño:

  1. Estética y Tema: El diseño debe ser moderno, limpio y acorde a los lineamientos básicos de usabilidad.
  2. Estructura Lógica: El sitio debe tener una jerarquía clara en su navegación.
  3. Uso de CSS3: Aplicación de técnicas avanzadas de CSS3, como bordes, sombras, gradientes, y transiciones para mejorar la experiencia del usuario.
