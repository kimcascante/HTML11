Guía de Estructura Lógica y Física

En este apartado se detalla cómo está organizada la estructura del proyecto tanto a nivel conceptual como técnico.

Estructura Lógica:

  1. Página de Requerimientos: Documento detallado de los requerimientos del cliente.
  2. Guía de Estructura Lógica y Física: Documento que muestra la estructura lógica y física del proyecto terminado.
  3. Carta de Presentación: Carta dirigida al cliente presentando el proyecto.
  4. Introducción: Página que plantea el contexto del proyecto y el problema presente.
  5. Diagnóstico y Recomendación: Análisis del problema y las soluciones propuestas.
  6. Conclusión: Página que evidencia el análisis y desarrollo del producto final.

Estructura Física:

  1. La estructura física del sitio web se compone de un archivo zip.
